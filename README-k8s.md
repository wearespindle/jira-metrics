# jira-metrics in k8s

## Settings

### JIRA

```txt
JIRA_HOST: https://domainname.atlassian.net
JIRA_USER: user@example.com
JIRA_PASS: SomePassword
```

### InfluxDB

```txt
INDLUXDB_HOST: host.fqdn.com
INDLUXDB_PORT: 8086
INDLUXDB_USER: jira
INDLUXDB_PASS: SomeOtherPassword
INDLUXDB_DATABASE: jira
```
